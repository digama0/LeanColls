    def List.zipWithAll {α : Type u_1} {β : Type u_2} {γ : Type u_3} (f : Option α → Option β → γ) :
    List α → List β → List γ

    Version of List.zipWith that continues to the end of both lists, passing none to one argument once the shorter list has run out.

    Equations
    • List.zipWithAll f [] x = List.map (fun (b : β) => f none (some b)) x
    • List.zipWithAll f (a :: as) [] = List.map (fun (a : α) => f (some a) none) (a :: as)
    • List.zipWithAll f (a :: as) (b :: bs) = f (some a) (some b) :: List.zipWithAll f as bs
